# java-rest-api-spring
Repositório dedicado aos estudos do curso “Spring Boot 3: API Rest em Java”. Aqui registro minha evolução no desenvolvimento de uma API REST completa usando Spring Boot 3, Java 17, JPA/Hibernate, Flyway, Lombok e MySQL, incluindo recursos essenciais como CRUD, validações, paginação, ordenação e boas práticas do ecossistema Spring.
